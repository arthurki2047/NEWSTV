# **App Name**: NewsTV

## Core Features:

- Live Channel Listing: Display a scrollable vertical list of live YouTube channels, showing channel logo, name, and a 'LIVE' indicator, optimized for small screens.
- Category Filtering: Allow users to navigate between different channel categories using the keypad's Left / Right buttons.
- YouTube Live Stream Player: Open a dedicated player page to play the selected YouTube live stream directly within the app when the 'OK' button is pressed.
- Fullscreen Video Playback: Provide an option for users to toggle fullscreen playback of the active YouTube live stream for an immersive viewing experience.
- Keypad Navigation: Enable intuitive navigation through the app's interface using 'Up', 'Down', 'Left', 'Right', 'OK' (Center), and 'Back' keypad buttons.
- Local Channel Data Management: Load and manage channel information (name, logo, YouTube URL, category) from a local JSON file to ensure fast access and offline capability.

## Style Guidelines:

- Primary action color: A clear, professional blue (#2E86C1) for active states and interactive elements, symbolizing clarity and technology.
- Background color: A very light, subtle blue-grey (#F2F7F9) providing a clean canvas that promotes readability and a lightweight feel.
- Accent color: A vibrant cyan (#33C6D2) to draw attention to critical information like the 'LIVE' indicator and provide a dynamic contrast.
- Body and headline font: 'Inter' (sans-serif) chosen for its exceptional legibility on small screens and a neutral, modern appearance, crucial for a KaiOS interface.
- Utilize minimalistic and universally recognizable icons, with a specific focus on a clear, high-contrast 'LIVE' indicator.
- Optimized layout for 240x320 screens, featuring a clear, vertically scrollable channel list and a full-screen capable video player.
- Minimal and subtle transitions, prioritizing performance and low memory usage inherent to the KaiOS platform.
